# Swasthya Saathi 360

AI powered rural healthcare platform for citizens, ASHA workers, doctors, and administrators.

## What’s included

- React + Vite + TypeScript frontend
- Express + Prisma + MySQL backend scaffold
- JWT auth and refresh token architecture
- Multilingual, mobile-first, PWA-ready UI
- AI integration surface for Gemini-based workflows

## Quick start

```bash
npm install
npm run dev
```

For the backend:

```bash
cd backend
npm install
npm run dev
```
